# dependency 설치
npm install [이름: express, cors, json, body-parser, nodemon]
- express : 백엔드 프레임워크
- cors : 프론트 백 설정
- json : json 파싱
- body-parser : 요청한 body 받기 
- nodemon : index.js 저장시 서버 새로고침
# package.json 추가설정
### type : module 
- node.js 의 모듈시스템을 ES6 를 사용할것입을 명시한다. require가 아닌(import, from)
